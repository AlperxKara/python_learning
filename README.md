## How to use
1. Clone repo: `git clone https://github.com/AlperxKara/python_learning.git`
2. Create virtualenv: `python3 -m venv venv && source venv/bin/activate`
3. Install deps (if any): `pip install -r requirements.txt`
4. Browse `notes/` and run scripts in `code/`.
